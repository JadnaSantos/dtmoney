## 💸 Dtmoney 

- ReactJS 
- TypeScript
- useState (hook que armazena dados que podem variar com o tempo, qualquer informação ou dado deve ser 
armazenados no useState, só assim será possível renderizar a informação)
- useEffect (hook utilizado para controlar a execução de uma função a partir da alteração de uma determinado
parâmetro)
- useContext (compartilhamento de estado entre vários components da aplicação)
- Styled Components (Biblioteca que permite escrever códigos CSS dentro do JavaScript)

## ⚙ Projeto

O dtmoney é uma aplicação de controle financeiro. Ao longo do projeto eu adicionei algumas funcionalidade adicionais, como:

- Armazenar os dados e qualquer conteúdo / dado salvo no objeto localStorage estará
disponível depois que o navegador for reiniciado (fechado e aberto novamente).

- Opção para deletar e renomear uma transação. Essa funcionalidade ainda está em desenvolvimento

- Adicionar mensagem flutuantes. Essa funcionalidade ainda está em desenvolvimento
